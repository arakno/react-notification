# REACT exercise
(for yara.com)
Build a reusable notification component that is triggered by a button.
this notification button should:
- appear as a toast (temp popup)
- last for 2 sec
- disappear

## Install
clone this repo then:
https://zeroserver.io/

`$ npm install -g zero`  

## Run (kordos!)
and start the server with:  
`$ zero`
